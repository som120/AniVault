# Contributing to AniFlux

Thank you for your interest in contributing to **AniFlux**!  
All contributions—bug reports, feature requests, documentation improvements, and code—are welcome.

By participating in this project, you agree to follow the project's
[Code of Conduct](CODE_OF_CONDUCT.md).

---

## Getting Started

- Make sure you have a GitHub account.
- Fork the repository and create a new branch for your changes.
- Keep your changes focused and well-documented.

---

## Reporting Bugs

If you find a bug:
1. Check existing issues to avoid duplicates.
2. Open a new issue with:
   - Clear steps to reproduce
   - Expected behavior
   - Actual behavior
   - Screenshots or logs if applicable
   - Device / Android version (if relevant)

---

## Feature Requests

Feature requests are welcome, but please:
- Explain the **problem** you are trying to solve
- Describe the **proposed solution**
- Mention any alternatives you’ve considered

Not all feature requests may be accepted, especially if they conflict with
project goals, API limitations, or Play Store policies.

---

## Pull Requests

Before submitting a pull request:
- Ensure your code builds successfully
- Keep PRs **small and focused**
- Write clear commit messages
- Reference related issues when applicable

Pull requests should:
- Follow the existing project structure
- Avoid unnecessary refactors
- Not include unrelated changes

---

## Code Style & Quality

- Follow existing formatting and naming conventions
- Prefer readable, maintainable code over clever solutions
- Avoid introducing new dependencies unless necessary
- Comment complex logic where appropriate

---

## API & Legal Notes

AniFlux is a third-party client application.

- Do **not** include API keys, secrets, or tokens in commits
- Respect the AniList API Terms of Service
- Do not add features that scrape content or bypass API limitations
- Avoid using copyrighted assets unless properly licensed

---

---

## Questions or Help

If you’re unsure about anything, feel free to:
- Open an issue for discussion
- Ask questions before starting large changes

Thank you for helping improve AniFlux!
